# data/

Generated data files — produced by the pipeline, not committed to Git.

## Files

| File | Generated by | Description |
|------|-------------|-------------|
| `crawler_output.jsonl` | `src/crawl/crawler.py` | Raw crawled pages (JSONL) |
| `preprocessed.jsonl` | `src/ie/preprocessing.py` | Cleaned text + sentences |
| `extracted_entities.csv` | `src/ie/ner_extraction.py` | NER-extracted entities |
| `extracted_relations.csv` | `src/ie/ner_extraction.py` | Extracted relations |
| `inferred_triples.csv` | `src/reason/swrl_rules.py` | SWRL-inferred triples |
| `kge/train.txt` | `src/kge/embedding.py` | KGE training split (80%) |
| `kge/valid.txt` | `src/kge/embedding.py` | KGE validation split (10%) |
| `kge/test.txt` | `src/kge/embedding.py` | KGE test split (10%) |

## samples/

Pre-generated sample files for demo and testing purposes.
These files are committed to Git and allow running the pipeline
without internet access or crawling.
